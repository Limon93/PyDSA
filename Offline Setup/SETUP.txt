A) Install NI-VISA and enable kernel modules

	1) cd into /rpm folder
	2) sudo yum install *.rpm
	3) sudo dkms autoinstall
	4) reboot


B) Install PyVISA and other software

	5) open terminal in /pip/setuptools folder
	6) sudo pip3 install *.whl
	7) unzip /pip/src/PyVISA-1.10.1.tar.gz 
	8) cd into extracted /PyVISA-1.10.1
	9) sudo python3 setup.py install
	
	10) cd into /pip/whl
	11) sudo pip3 install *.whl

	12) plug usb ds1054z
	13) sudo rmmod usbtmc
	14) sudo NIvisaic
	15) USB0::xxxxxx::xxxxx::xxxxxxxx should appear


C) Use software PyDSA

	16) sudo python3 PyDSA.py
	17) If you got a small screen to display full PyDSA window:
	    change gnome font scaling factor in Applications -> Accessories -> Tweaks -> Fonts -> Scaling Factor
